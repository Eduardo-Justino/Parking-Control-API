
# Parking-Control-API


### Métodos: 

#### - Cadastrar
#### - Buscar todos
#### - Buscar por id
#### - Deletar por id
#### - Atualizar cadastro por id

# Validações: 

#### Cadastro : Uma vaga por bloco e apartamento 
#### Cadastro : A vaga tem que estar disponivel 
#### Cadastro : Placa do automovel cadstrada tem que ser unica  
#### Cadastro : Nenhum campo pode estar vazio
